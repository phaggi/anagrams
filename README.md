# anagrams
Program for find and print an anagrams for noun word.<br>
###Requirements:

    python~=3.7
    toml~=0.10.2
    future~=0.18.2

For install unpack 'russian_nouns.zip' 
to root of project.<br>
See at anagrams.py examples for use.

    - find_anagrams('столик')
    - find_anagrams()
    - find_anagrams(cycle=True)
